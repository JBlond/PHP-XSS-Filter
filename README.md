# PHP-XSS-Filter

## Install

```BASH
composer require jblond/php-xss-filter
```

## Example
```PHP
	require './xss_filter.class.php';
	$xss = new xss_filter();
	$string = '<iframe>blah';
	$string = $xss->filter_it($string );
	echo $string;
```
